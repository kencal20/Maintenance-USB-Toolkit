====================================================
            Maintenance Toolkit USB
====================================================

Owner: kencal20
Version: 1.0
Purpose: Cross-platform basic diagnostics & utilities

----------------------------------------------------
OVERVIEW
----------------------------------------------------
This USB contains lightweight maintenance tools for:

• Windows diagnostics
• Linux system inspection
• Portable utilities
• Custom scripts

No installation required.
All tools run portable.

----------------------------------------------------
FOLDER STRUCTURE
----------------------------------------------------

Maintenance_Tools/
│
├── launch_linux.sh       → Linux launcher menu
├── launch_windows.bat    → Windows launcher menu
├── Sysinternals/         → Windows system tools
├── Scripts/              → Custom scripts
├── Other_Tools/          → Portable utilities

----------------------------------------------------
USAGE
----------------------------------------------------

On Linux:
  1. Open terminal
  2. Navigate to this folder
  3. Run:
     ./launch_linux.sh

On Windows:
  Double-click:
     launch_windows.bat

Some tools may require Administrator privileges.

----------------------------------------------------
RECOMMENDED WORKFLOW
----------------------------------------------------

1. Run system information
2. Check disk health
3. Inspect running processes
4. Review startup programs
5. Use portable tools if needed

----------------------------------------------------
NOTES
----------------------------------------------------

• This USB is formatted for cross-platform use.
• No system files are modified automatically.
• Always verify drive letters before running disk tools.
• Keep tools updated periodically.
